# システム概要

## 背景

日本は石油の 97% を輸入に依存し、IEA 基準で 90 日分の備蓄義務を負うエネルギー安全保障の脆弱国である。経済産業省は「石油備蓄の現況」速報を日次 PDF で公表しているが、その存在自体が一般に認知されておらず、また「215 日分」のような数値単体では多寡の比較基準を持てないため、ニュース感度の高い層であっても「いま日本の備蓄が安全圏にあるのか」を肌感覚として把握できない。SNS で備蓄・シーレーン関連の話題が断続的に拡散されるたびに、信頼できる軽量な参照源が不在のまま、出典不明の数字が独り歩きする状況が繰り返されている。

## 目的

経産省の速報 PDF を自動取得・パースし、「日本の石油備蓄、いま N 日分」という単一指標を訪問から数秒以内に把握できる公開サイトを提供する。総量を秒単位で減算するカウンターと、過去最高値に対する充填率を示すタンクゲージで「多いか少ないか」の比較基準を与え、12 ヶ月推移と日本周辺タンカー集計で動向の文脈を補う。SNS シェア（X / リンクコピー / OGP）を主要動線として、第三者が自発的に出典付きで拡散できる構造を実現することを目的とする。
