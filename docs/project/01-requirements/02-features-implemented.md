# 実装済み機能一覧

| 機能ID | Category 1 | Category 2 | 機能名 | 説明 |
|--------|-----------|-----------|--------|------|
| FN-001 | カウンター表示 | リアルタイム | 備蓄日数カウンター | 総備蓄日数を「N 日 HH 時間 MM 分 SS 秒」で表示し、1 秒ごとに減算する |
| FN-002 | カウンター表示 | ビジュアル | タンクゲージ | 過去最高値（`PEAK_REFERENCE`）に対する充填率を色分けで表示 |
| FN-003 | カウンター表示 | ビジュアル | 12 ヶ月推移グラフ | 直近 12 ヶ月の総備蓄日数推移をライン表示、ホバー/フォーカスで詳細 |
| FN-004 | カウンター表示 | 詳細 | 3 区分内訳トグル | 国家・民間・産油国共同備蓄の内訳をグラフ上で切替表示 |
| FN-005 | カウンター表示 | 補助 | 出典・データ時点フッター | 経産省 PDF リンク、公表日、`asOf` を常時表示 |
| FN-006 | カウンター表示 | 補助 | 備蓄データ古さ警告 | `asOf` から 14 日以上経過時に警告バナー表示 |
| FN-007 | タンカー表示 | 集計 | 海域内タンカー隻数表示 | 日本周辺海域でサンプリングしたタンカー隻数を表示 |
| FN-008 | タンカー表示 | 集計 | 日本港向けタンカー集計 | `destination` が日本港のタンカー隻数と上位港を表示 |
| FN-009 | タンカー表示 | 補助 | タンカーデータ古さ警告 | 6 時間以上 `tankers.json` が更新されない場合に警告 |
| FN-010 | シェア | SNS | X シェアボタン | 「日本の石油備蓄、いま N 日分。」を X で prefill 共有 |
| FN-011 | シェア | SNS | リンクコピー | 同テキストと URL をクリップボードにコピー |
| FN-012 | データ自動更新 | 備蓄 | 備蓄 PDF 日次取得 | GitHub Actions が毎朝 07:00 JST に経産省 PDF を取得・パース・`snapshots.json` 更新 |
| FN-013 | データ自動更新 | 備蓄 | PDF バリデーション | 範囲（50〜500 日）・内訳合計（±2）・前日比（±30 日）の閾値チェックで異常値を排除 |
| FN-014 | データ自動更新 | タンカー | AIS タンカー毎時取得 | aisstream.io WebSocket を 8 分サンプリングして集計し `tankers.json` 更新 |
| FN-015 | データ自動更新 | 共通 | 失敗時のデータ保持 | 取得失敗時は古い JSON を保持し、警告のみ表示（壊れたデータは出さない） |
| FN-016 | 品質保証 | テスト | パーサ・集計のユニットテスト | `scripts/test_*.py` + `test.yml` で自動実行 |
| FN-017 | SEO・配布 | 検索 | robots.txt / sitemap.xml | クローラ向け基本設定 |
