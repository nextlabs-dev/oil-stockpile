## メモ

- US-001 / US-004 / US-006 はカウンタータブのファーストビューを構成する中核ストーリー
- US-002 はシェアによる拡散動線、US-007 はデータ信頼性維持の観点で High
- US-005 は独自価値だが、メインの「備蓄日数」とは別文脈のため Medium
