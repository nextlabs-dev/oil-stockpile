# 非機能要件一覧

| カテゴリ | 要件 | 説明 |
|----------|------|------|
| セキュリティ | API キー管理 | `AISSTREAM_API_KEY` は GitHub Secrets で管理し、クライアントコードに含めない |
| セキュリティ | 個人情報 | 個人情報・ログイン情報を一切扱わない（収集・保存しない） |
| 可用性 | データ整合性 | 取得失敗時は古い JSON を保持し、壊れた値を表示しない |
| ユーザビリティ | レスポンシブ | 375px のモバイル幅からデスクトップまでレイアウト破綻なし |
| ユーザビリティ | 対応ブラウザ | Chrome / Safari / Edge / Firefox の最新 2 バージョン |
